FAQ

What platforms are supported?

Windows, Mac and Linux. Generally any platform that Adobe Air runs on.

What makes Vendetta unique?

Vendetta is a modern text editor; Its inspired by the simplicity and power of of command line editors such as Vim and GNU Emacs, but without the historical baggage.

However, Vendetta is not a me-too editor, its a test-bed for some radical changes to the traditional editor user interface.

What technologies are behind Vendetta?

Vendetta is part of the vanguard of a new wave desktop software, developed using web technologies. Vendetta is written in Javascript, CSS and HTML, making it easy to modify and extend. The Adobe Air hosts the application, and provides an abstraction layer, hiding operating system specifics.

Vendetta uses a number of Javascript libraries, including jQuery, Backbone.js and Require.js. The central editor component is the excellent ACE library by Ajax.org.

How is Vendetta licenced

Vendetta is free & open source. It is distributed under the GNU GPL licence.
